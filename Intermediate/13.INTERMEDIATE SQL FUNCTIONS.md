# 🔢 INTERMEDIATE SQL FUNCTIONS

This file covers three important SQL function categories:

1. Numeric Functions  
2. Conversion Functions  
3. NULL Handling Functions  

These are widely used in data cleaning, reporting, and interviews.

---

# 🔢 1. NUMERIC FUNCTIONS

Numeric functions perform mathematical operations on numbers.

## ABS()
Returns absolute value.

SELECT ABS(-10);  -- 10

---

## ROUND()
Rounds a number to specified decimal places.

SELECT ROUND(12.5678, 2);  -- 12.57
SELECT ROUND(12.4);        -- 12
SELECT ROUND(12.6);        -- 13

---

## CEIL() / CEILING()
Rounds a number UP.

SELECT CEIL(12.1);  -- 13

---

## FLOOR()
Rounds a number DOWN.

SELECT FLOOR(12.9);  -- 12

---

## POWER()
Raises a number to exponent.

SELECT POWER(2, 3);  -- 8

---

## SQRT()
Returns square root.

SELECT SQRT(25);  -- 5

---

## Use Cases
- Salary rounding
- Financial calculations
- KPI adjustments
- Discount computation

---

# 🔄 2. CONVERSION FUNCTIONS

Used to convert data types.

## CAST()
Explicit type conversion.

SELECT CAST('123' AS INT);
SELECT CAST(123 AS VARCHAR);
SELECT CAST('2024-01-01' AS DATE);

---

## CONVERT() (SQL Server)
SELECT CONVERT(INT, '456');

---

## IMPLICIT CONVERSION
Automatic conversion by SQL engine.

SELECT '10' + 5;

---

## Use Cases
- Fix mismatched join columns
- Clean raw data
- Standardize data formats
- Fix calculation errors

---

# ❌ 3. NULL HANDLING FUNCTIONS

NULL = missing or unknown value.

## IS NULL / IS NOT NULL
SELECT * FROM employees WHERE salary IS NULL;

---

## COALESCE()
Returns first non-null value.

SELECT COALESCE(NULL, NULL, 100, 200);  -- 100
SELECT COALESCE(bonus, 0) FROM employees;

---

## NULLIF()
Returns NULL if both values are equal.

SELECT NULLIF(10, 10);  -- NULL
SELECT NULLIF(10, 5);   -- 10

---

## IFNULL() (MySQL)
SELECT IFNULL(salary, 0);

---

## NVL() (Oracle)
SELECT NVL(salary, 0);

---

## Use Cases
- Replace NULL values
- Avoid divide-by-zero errors
- Clean missing data
- Improve reporting accuracy
